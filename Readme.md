# Velosofize docs (Deprecated in favor of [Velo-Sofia](https://velo-sofia.com) )

This repo contains the configuration and information for the blog section of [Velosofize](https://docs.old.velo-sofia.com) and uses [this custom theme](https://github.com/velosofist/velosofize-minima-theme/). The project is deprecated due to low interest in contributions, so I can focus on the more maintainable [Velo-Sofia](https://velo-sofia.com) instead.

Contributions with useful information or edits are welcome, though truly open-sourcing the blog under a permissive license is tricky due to the usage of quotes and materials from external sources, including government agencies.
